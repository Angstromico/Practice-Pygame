# Pygame Practice Repository

This is a practice repository for learning and experimenting with Pygame, a popular Python library for creating video games and multimedia applications.

## About

This repository contains simple Pygame projects and exercises designed to help understand the fundamentals of game development in Python. It's perfect for beginners who want to learn Pygame through hands-on practice.

## Current Project

The main project (`main.py`) is a basic Pygame window setup that demonstrates:

- Initializing Pygame
- Creating a display window
- Basic event handling for window closing

## Requirements

- Python 3.7 or higher
- Pygame 2.6.1

## Installation

1. Clone this repository:

```bash
git clone <repository-url>
cd pygame
```

2. Install the required dependencies:

```bash
pip install -r requirements.txt
```

## Usage

Run the main Pygame application:

```bash
python main.py
```

This will open an 800x600 window that you can close by clicking the X button.

## Learning Goals

- Understanding Pygame initialization and setup
- Managing game loops and events
- Working with display surfaces
- Handling user input
- Creating simple graphics and animations
